Sprites véhicules (mode "1 direction + rotation runtime")

- Chaque fichier est une seule vue "Nord" (le véhicule regarde vers le haut).
- Le moteur tourne l'image en fonction de Vehicle.angle (cf. js/render/VehicleSpriteBank.js).
- Fond transparent (PNG RGBA).

Chemin:
  assets/vehicles/<ID>.png
Ex:
  assets/vehicles/COPCAR.png
